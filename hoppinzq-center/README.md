### hoppin的注册中心  
#### 须知

- 每次都必须先启动注册中心模块
- 其他模块需要配置注册中心的地址：
```yaml
zq-server:
  name: 模块名
  username: zhangqi # 用户名，写死
  password: 123456 # 密码，写死
  prefix: /service # 服务前缀，写死
  ip: 127.0.0.1 # 服务所在的地址，写外网IP
  center-addr: http://127.0.0.1:8801/service # 注册中心地址
```
即可将模块注册到注册中心，这样其他服务就能调用其他服务注册的rpc接口了。  
#### 注意
这个服务是强制的，因为每个服务（包括AI服务)的鉴权，都必须要走auth服务，必须通过本框架提供的rpc的方式来进行。
