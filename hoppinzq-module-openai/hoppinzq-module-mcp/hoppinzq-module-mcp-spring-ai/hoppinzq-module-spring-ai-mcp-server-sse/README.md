### 基于sse的最基本的MCP Server

### 用前须知

#### 你需要有一个开源的工作流项目，可以微信：zhangqiff19 联系我获取

### 如何使用

#### 1、启动该项目。

#### 2、在Cherry Studio里选择MCP设置，然后添加一个MCP服务器。传输方式选择`sse`，地址填写`http://localhost:9000/sse` ，注意端口号`9000`是配置文件里配置的。

#### 3、开始对话。示例提示词：

- AI，你知道哪些网站
- AI，打开张祺的网站
- AI，帮我发起一个请假流程
- AI，我想请一天病假，再请5天事假，理由是旅游，两个流程审批人都是admin
- AI，帮我看看我的待办有什么
- AI，审批我的第一个待办
- AI，审批我的所有待办
